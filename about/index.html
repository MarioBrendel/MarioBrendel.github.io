Hi, I’m Mario Brendel, the Founder of this blog. My goal is to teach you the fundamentals about the Java EE stack.
Currently I’m trying to get into the stack myself, so most of my tutorials will be from the view of a beginner. The main goal is to just focus on the really important aspects of a framework and show you via some code examples, how you can use the framework yourself. Before each tutorial series i will give a general overview and after that we will jump right into it. I believe, that showing code examples will be the quickest way to get you fit with the technology. After a series is finished, i will program a little application that uses this new framework. You can download the source code via Github: https://github.com/MarioBrendel.


Since I’m not a native english speaker, there may be some grammatical errors. So if there are some ludicrous errors you can always correct me :).
Feel free to mail me whenever you have suggestions or other things that are on your mind.
